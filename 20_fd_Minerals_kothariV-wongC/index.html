Key Features of Foundation

Mobile First

Customizable

Faster For Users

Navigable Website

1. Mobile First

You can build pages catered for small devices first, and them increase complexity as devices get larger for a complete responsive design.

2. Customizable

You can customize the smallest of elements in sites like never before, such as the size of columns, colors, fonts, etc.

3. Faster For Users

Due to having a faster core, Foundation allows you to create animations wth relatively fast loading soeeds and a simpler, cleaner, and faster visual aesthetic.

4. Navigable

The website is easy to navigate and contains hundreds of building blocks for you to get started learning Foundation.
